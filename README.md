# Unfat

Unfat helps you quickly track what you eat and drink, how you sleep, and your weight over time.

The focus is low friction:

- snap a photo of food/drink
- submit immediately
- let AI fill nutrition values in the background
- keep moving

## What You Can Track

### Food and Drink

Each entry can include:

- photo (camera or upload)
- timestamp
- notes
- nutrition values (manual or AI-estimated)

Nutrition includes calories, salt, sugar, fibre, caffeine, selected minerals, and selected vitamins.

### Sleep

- start time
- end time
- sleep score

### Weight

- weight value
- timestamp

## How Food Photo AI Works

1. You create a food entry.
2. The entry appears right away.
3. AI analysis runs in the background.
4. The table updates automatically when results arrive.

If analysis fails, you can tap **Retry AI**.
If you edit values and want the original AI values back, tap **Revert to AI estimate**.

## Multi-User and Admin Controls

Unfat supports multiple users.

Admins can:

- create users
- reset passwords
- disable/enable users
- delete users and all their related data
- update AI service settings

Important: password reset is admin-only (there is no self-service reset flow).

## Day-to-Day Use

### Log a Meal

- Open **Food**
- Tap the `+` button
- Take a photo (or upload one)
- Submit

### Log Sleep

- Open **Sleep**
- Tap the `+` button
- Enter start/end + score
- Save

### Log Weight

- Open **Weight**
- Tap the `+` button
- Enter value + time
- Save

### View Trends

- Open **Charts**
- Choose a date window (7d, 15d, 30d, 90d, or custom)
- Compare food, sleep, and weight trends

## Time and Timezone

- Date/time fields default to “now”.
- Times are shown in your browser’s local timezone.
- Photo entries use the photo’s taken time when available.

## Mobile and Desktop

- Mobile-first interface
- installable as a web app
- desktop supports drag-and-drop image upload
- shows an offline message if your connection drops

## First Login

On first run, the default admin account is:
